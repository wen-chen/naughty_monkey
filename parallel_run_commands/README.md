# Parallel run commands

```
usage: parallel_run_commands.py [-h] -c CMD_FILE_NAME [-t THREAD_NUM]

optional arguments:
  -h, --help            show this help message and exit
  -c CMD_FILE_NAME, --cmd CMD_FILE_NAME
                        Command list file
  -t THREAD_NUM, --thread_num THREAD_NUM
                        Number of threads (default 2)
```
